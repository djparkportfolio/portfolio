Nest is a rocking chair and fool rest set encouraging a parent and a child to sit and spend time together. The initial sketch was inspired by the theme of Easter to bring the fun memory of the childhood that everyone cherishes. Nest was built into a life-size scale with birch plywood. The cushion was fabricated with PE foam into hand sewn fabrics.
